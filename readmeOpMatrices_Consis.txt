# Programa de Operaciones con Matrices

Este es un programa en C que permite realizar diversas operaciones matriciales, como suma, resta, multiplicación, multiplicación de un vector por una matriz y suma de columnas. Además, el programa ofrece la capacidad de guardar y cargar datos desde archivos para mantener la persistencia de datos entre ejecuciones.

## Características

- Realiza operaciones matriciales básicas: suma, resta y multiplicación.
- Permite la multiplicación de un vector por una matriz.
- Calcula la suma de columnas de una matriz seleccionada.
- Guarda y carga matrices y un vector desde y hacia archivos.
- Facilita la elección de cargar datos existentes o proporcionar nuevos datos al usuario.

## Uso

1. Ejecuta el programa.
2. Selecciona el número de filas y columnas para las matrices.
3. Elije si deseas cargar datos desde un archivo o proporcionar nuevos datos.
4. Realiza la operación deseada (suma, resta, multiplicación, multiplicación de un vector por una matriz o suma de columnas).
5. Los resultados se mostrarán en pantalla y, si corresponde, se guardarán en archivos.

## Requisitos

- Compilador C compatible (por ejemplo, GCC).
- Acceso de escritura en el directorio donde se encuentra el programa para guardar datos en archivos.

## Archivos de Datos

Los datos se guardarán en los siguientes archivos por defecto:

- Matrices: "InfoMat.txt"
- Vector: "InfoVec.txt"

Asegúrate de tener permisos de escritura en el directorio para guardar los datos correctamente.

## Ejemplo de Archivo de Datos

El formato de los archivos de datos es el siguiente:

Matriz A:
1 2 3
4 5 6
7 8 9

Matriz B:
9 8 7
6 5 4
3 2 1

Vector:
1 2 3
